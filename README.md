# Company Policy AI Assistant

An AI-powered assistant that answers employee questions from company policy documents using Retrieval-Augmented Generation (RAG).# Company Policy AI Assistant

An AI-powered assistant that answers employee questions from company policy documents using Retrieval-Augmented Generation (RAG).

## Features
- PDF ingestion
- Text chunking
- OpenAI embeddings
- FAISS vector search
- GPT-based Q&A chatbot

## Tech Stack
Python, LangChain, OpenAI API, FAISS

## How It Works
PDFs → Text Splitting → Embeddings → FAISS Vector DB → GPT Response

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Set OpenAI API key:
   setx OPENAI_API_KEY "your_key_here"

3. Run ingestion:
   python scripts/ingest.py

4. Start chatbot:
   python scripts/chat.py
